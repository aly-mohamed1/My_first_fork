# Practice Questions: PHP Control Structures (If-Conditions & Loops)

*   Even or Odd: Write a program that checks if a given number is even or odd using an if-else statement and the modulus operator %.

*   Leap Year Checker: Create a script to determine if a specific year is a leap year. (Logic: A year is leap if divisible by 4, but not by 100, unless it is also divisible by 400).


*   Largest of Three: Use nested if-else statements to find the largest among three different numbers.
*   Simple Calculator: Use if-elseif-els to design a calculator that takes two numbers and an operator (+, -, *, /) and displays the result.
*   Sum of Numbers: Use a for loop to calculate the sum of the numbers from 1 to 100.

*   Fibonacci Series: Use a loop to print the first N terms of the Fibonacci sequence, where each number is the sum of the two preceding ones.

*   Pattern Printing: Use nested loops to print a pyramid or triangle pattern of stars (`*`) based on a height defined by the user.

*   Array Filtering: Create an array of numbers and use a foreach loop with an if condition to print only the numbers greater than a specific threshold (e.g., 50).
